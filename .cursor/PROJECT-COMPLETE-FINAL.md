## 🤖 OpenAI Integration

**Powered by:** GPT-4 Turbo

**Setup:**
```bash
# .env.local
OPENAI_API_KEY=sk-your-key-here
OPENAI_MODEL=gpt-4-turbo  # or gpt-3.5-turbo
```

**Cost Estimates:**
- Light use: ~$75-150/month (GPT-4)
- Budget option: ~$7.50/month (GPT-3.5)

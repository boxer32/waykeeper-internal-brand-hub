## 🚀 How to Use

### **1. Access the Hub**
```
http://localhost:3001
```

### **2. Enable AI Features**
```bash
# Edit .env.local
OPENAI_API_KEY=sk-your-actual-key-here

# Restart server
npm run dev
```

### **3. Switch Language**
- Click "ไทย" or "EN" in header
- Navigation switches instantly
- Preference saved automatically
